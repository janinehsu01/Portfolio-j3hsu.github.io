Definitions

Define a table
A table in html is defined by a <table> tag and uses rows and columns to present tabular data
<tr> indicates how many rows the table has
<th> indicates the table header
<td> indicats how many cells each row of the table has
<thead> indicates the head content of the table
<tfoot> indicates the footer content of the table
<caption> is used to add a caption to the top of the table
Difference between colspan and rowspan
Colspan indicates how many columns the cells in the table need to span
Rowspan indicates how many rows the cells in the table need to span
Difference between cells spacing and padding
Cells spacing is the space between each cell in the table
Cells padding is the space between the cell content and the cell edge
Define table borders using CSS
Use the border property to indicate what type of border it is and how thick and thin the border is
Control border widths using CSS
Use the width property to change the width of the border
